# ⚡ PromptForge
 
An AI Prompt Engineering Platform that turns rough ideas into structured, high-quality AI prompts.
 
Built for **AI Engineer Round 2 Technical Assignment**.
 
---
 
## What It Does
 
You type a basic requirement → PromptForge returns a fully structured prompt with a quality score out of 100.
 
---
 
## Pages
 
- **Dashboard** — Platform overview and prompt engineering principles
- **Prompt Generator** — Enter a requirement, get a structured prompt with quality score
- **Iteration Lab** — Before/after examples showing how prompts improve
- **Content Studio** — Templates for Video, Image, and Website prompts
- **Saved Prompts** — Your personal prompt library saved to database
---
 
## Tech Stack
 
- React + TypeScript + Vite
- TailwindCSS
- Supabase (PostgreSQL)
---
 
## Getting Started
 
```bash
git clone https://github.com/your-username/promptforge.git
cd promptforge
npm install
```
 
Create a `.env` file:
 
```env
VITE_SUPABASE_URL=your-supabase-url
VITE_SUPABASE_ANON_KEY=your-anon-key
```
 
```bash
npm run dev
```
